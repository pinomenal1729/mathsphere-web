📋 JSON Format Guide
Use this schema when preparing bulk imports
[ { "exam": "jam", // REQUIRED: "jam" | "gate" | "csir" "year": 2023, // Optional: integer year "paper": "Session 1", // Optional: paper identifier "topic": "Real Analysis",// Optional: mathematical topic "subtopic": "Continuity", // Optional: subtopic "difficulty": "moderate", // Optional: "easy"|"moderate"|"difficult" "source": "Official IIT JAM 2023", "question": "Let \\( f: \\mathbb{R} \\to \\mathbb{R} \\) ...", "options": [ // Array of 4 options "A) option text", "B) option text", "C) option text", "D) option text" ], "answer": "B) option text", "explanation": "Step-by-step explanation...", "marks": 2.0, // Optional: marks for this question "negative": 0.67, // Optional: negative marking "q_type": "mcq", // "mcq"|"msq"|"nat"|"subjective" "tags": ["integration", "Riemann"] } ]
💡 Tips for Pasting Paper Text

✅ Works best when: Questions are numbered (Q1, Q.1, 1.), options are labeled A/B/C/D

✅ LaTeX: If your paper has LaTeX, keep it — AI will preserve it

✅ Large papers: Paste the full paper — long texts are split automatically

⚠️ PDF text: Copy-paste from PDF works; scanned PDFs may need OCR first

⚠️ Review: AI extraction is ~90% accurate — always browse and verify after import

⚠️ Duplicates: Currently no duplicate detection — avoid importing the same paper twice
